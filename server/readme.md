# server側

## 機能
サインアップとログインの機能を作る

## 使った技術
- Node.js (jsでサーバを作る環境)
- typescript (jsの型宣言)
- Express (webアプリケーションのフレームワーク)
- bycrypt (パスワードを暗号化)
- postgrelsql (sqlデータベース)
- pg (postgrelsqlと連携)
- jsonwebtoken (アクセス権限(トークン))
- dotenv (見られたくない.envファイルと繋ぐ)
- nodemon (自動でリロード)
- expresss-validator (リクエストに問題があるかの確認)
